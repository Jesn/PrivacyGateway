🚀 项目介绍

Privacy Gateway 是一个专注于隐私保护的反向代理服务，通过过滤敏感的 HTTP 头信息来保护用户隐私。它支持 HTTP 和 WebSocket 代理，具有轻量级、高性能的特点。

🔒 隐私保护

自动过滤 CF-Ray、X-Forwarded-For 等敏感头信息

⚡ 高性能

基于 Go 原生 HTTP 库，支持并发处理

🌐 双协议支持

同时支持 HTTP 和 WebSocket 代理

🐳 容器化部署

Docker 多阶段构建，极简生产镜像

📡 API 端点

HTTP 代理
GET/POST/PUT/DELETE /proxy?target=<目标URL>[&proxy=<代理地址>]

通过代理访问任何 HTTP 服务，自动过滤敏感头信息，支持上游代理

WebSocket 代理
WebSocket /ws?target=<目标WebSocket地址>[&proxy=<代理地址>]

代理 WebSocket 连接，支持双向实时通信和上游代理

高级代理配置
Header: X-Proxy-Config: <Base64编码的JSON配置>

支持复杂代理配置，包括认证、超时等参数

🔧 X-Proxy-Config 详细说明

X-Proxy-Config 是一个强大的功能，允许您通过 HTTP 头传递复杂的代理配置。配置以 JSON 格式编写，然后进行 Base64 编码。

配置参数说明

{ "url": "代理服务器地址 (必需)", "auth": { "username": "用户名 (可选)", "password": "密码 (可选)" }, "timeout": "超时时间，单位秒 (可选，默认30)", "headers": { "自定义头名": "自定义头值 (可选)" } }

支持的代理协议

🌐 HTTP/HTTPS

http://proxy.example.com:10805
https://proxy.example.com:10805

🔒 SOCKS5

socks5://proxy.example.com:1080
支持用户名密码认证

JavaScript 完整示例

// Privacy Gateway 代理配置工具类 class PrivacyGatewayClient { constructor(gatewayUrl = 'http://localhost:10805') { this.gatewayUrl = gatewayUrl; } createProxyConfig(config) { if (!config.url) { throw new Error('代理 URL 是必需的'); } const proxyConfig = { url: config.url, timeout: config.timeout || 30, ...config }; return btoa(JSON.stringify(proxyConfig)); } async request(targetUrl, options = {}) { const { method = 'GET', headers = {}, body, proxyConfig } = options; const url = new URL('/proxy', this.gatewayUrl); url.searchParams.set('target', targetUrl); const requestHeaders = { ...headers }; if (proxyConfig) { requestHeaders['X-Proxy-Config'] = this.createProxyConfig(proxyConfig); } try { const response = await fetch(url.toString(), { method, headers: requestHeaders, body: body ? JSON.stringify(body) : undefined }); return response; } catch (error) { console.error('代理请求失败:', error); throw error; } } createWebSocket(targetUrl, proxyConfig = null) { const url = new URL('/ws', this.gatewayUrl.replace('http', 'ws')); url.searchParams.set('target', targetUrl); if (proxyConfig && proxyConfig.url) { url.searchParams.set('proxy', proxyConfig.url); } return new WebSocket(url.toString()); } } // 使用示例 const client = new PrivacyGatewayClient(); // 1. 简单请求 const response = await client.request('http://httpbin.org/get'); // 2. HTTP 代理 const proxyConfig = { url: 'http://proxy.example.com:10805', timeout: 60 }; await client.request('http://ipinfo.io/json', { proxyConfig }); // 3. SOCKS5 代理 + 认证 const socksConfig = { url: 'socks5://proxy.example.com:1080', auth: { username: 'user', password: 'pass' }, timeout: 120 }; await client.request('https://api.github.com/user', { headers: { 'Authorization': 'token your_token' }, proxyConfig: socksConfig }); // 4. WebSocket 代理 const ws = client.createWebSocket('wss://echo.websocket.events', socksConfig); ws.onopen = () => ws.send('Hello!'); ws.onmessage = (event) => console.log(event.data); // 5. 批量请求 const urls = ['http://httpbin.org/get', 'http://httpbin.org/ip']; const promises = urls.map(url => client.request(url, { proxyConfig })); const responses = await Promise.all(promises);

💡 使用示例

基础 cURL 请求示例

# 直接代理（无上游代理） curl "http://localhost:10805/proxy?target=http://ipinfo.io/json" # 通过 HTTP 代理 curl "http://localhost:10805/proxy?target=http://ipinfo.io/json&proxy=http://proxy.example.com:10805" # 通过 SOCKS5 代理 curl "http://localhost:10805/proxy?target=http://ipinfo.io/json&proxy=socks5://proxy.example.com:1080"

cURL 高级代理配置示例

# 带认证的代理配置 proxy_config='{"url":"http://proxy.example.com:10805","auth":{"username":"user","password":"pass"},"timeout":60}' encoded_config=$(echo -n "$proxy_config" | base64) curl -H "X-Proxy-Config: $encoded_config" \ "http://localhost:10805/proxy?target=http://ipinfo.io/json" # SOCKS5 代理配置 socks_config='{"url":"socks5://proxy.example.com:1080","auth":{"username":"user","password":"pass"},"timeout":120}' encoded_socks=$(echo -n "$socks_config" | base64) curl -H "X-Proxy-Config: $encoded_socks" \ -H "User-Agent: PrivacyGateway/1.0" \ "http://localhost:10805/proxy?target=https://api.github.com/user"

WebSocket 代理示例

# 直接 WebSocket 代理 wscat -c "ws://localhost:10805/ws?target=wss://echo.websocket.events" # 通过上游代理的 WebSocket wscat -c "ws://localhost:10805/ws?target=wss://echo.websocket.events&proxy=socks5://proxy.example.com:1080" # 简单的 JavaScript WebSocket 代理 const ws = new WebSocket('ws://localhost:10805/ws?target=wss://echo.websocket.events&proxy=http://proxy.example.com:10805'); ws.onopen = () => console.log('WebSocket 连接已建立'); ws.onmessage = (event) => console.log('收到消息:', event.data); ws.onerror = (error) => console.error('WebSocket 错误:', error); ws.onclose = () => console.log('WebSocket 连接已关闭');
